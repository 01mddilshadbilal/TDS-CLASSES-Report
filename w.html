1. Introduction & Project Overview

TDS CLASSES is a comprehensive Android application designed for coaching institutes and private tutors to manage students, attendance, learning schedules, and performance tracking efficiently.

The application provides a digital transformation solution that replaces traditional paper-based systems with an intelligent, automated platform.

Key Objectives:

  • Digitize student enrollment and profile management
  • Automate attendance with color-coded system
  • Implement intelligent learning time reminders
  • Provide data persistence and security
  • Enable easy data sharing and reporting
  • Offer a professional user experience

7.2 Learning Time Alarm System

Alarm Workflow:

  1. Mark student as PRESENT
  2. Check if Alarm Manager is ON
  3. Get student's Daily Learning Time
  4. Schedule alarm: Present Time + Learning Time
  5. Alarm triggers with notification
fun scheduleLearningAlarm(student: Student) { val alarmTime = System.currentTimeMillis() + (student.dailyLearningTime * 60 * 1000) val intent = Intent(context, AlarmReceiver::class.java).apply { putExtra("studentName", student.name) } val pendingIntent = PendingIntent.getBroadcast( context, student.id.hashCode(), intent, PendingIntent.FLAG_IMMUTABLE ) alarmManager.setExact( AlarmManager.RTC_WAKEUP, alarmTime, pendingIntent ) }
